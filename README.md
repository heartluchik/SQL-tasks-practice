# SQL Tasks

This repo contains SQL/PLSQL tasks from a technical test.

## Task 1 — SQL queries (employee + department)

Queries using joins, aggregation, and window functions (RANK) to find:
- employees who earn more than their manager
- top salary in each department
- departments with many high earners
- other department/salary summaries

## Task 2 — Call log table + loading procedure

Oracle table design + PL/SQL procedure to load data from a raw log table into a clean table:
- constraints and indexes
- batch insert (FORALL)
- basic error handling
- lookup of contragents by phone number
